## 一般问题

### 什么是 Onlook？

Onlook 是一个开源的"设计师版 Cursor"，让设计师能够直接在浏览器 DOM 中对 React 和 TailwindCSS 项目进行实时编辑。它提供了设计与开发之间的无缝集成。

### Onlook 是免费使用的吗？

是的，Onlook 是完全开源且免费使用的。您可以从 [GitHub](https://github.com/onlook-dev/onlook) 克隆仓库并在本地运行。

### 我可以用 Onlook 构建哪些类型的项目？

Onlook 适用于任何 React 和 TailwindCSS 项目。您可以构建网站、Web 应用程序、仪表板等。

## 技术问题

### Onlook 使用了哪些技术？

Onlook 构建于：

- Next.js
- Drizzle
- Supabase
- Bun

### 我可以在现有项目中使用 Onlook 吗？

是的，Onlook 可以在任何 Next.js + TailwindCSS 项目上运行。您可以导入项目并立即开始编辑。

### Onlook 与类似工具相比如何？

Onlook 是 Bolt.new、Lovable、V0 和 Figma Make 等工具的开源替代方案，具有独特的功能和设计到代码工作流方法。
