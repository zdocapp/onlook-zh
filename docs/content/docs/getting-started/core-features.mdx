## 可视化编辑器

可视化编辑器允许您像 Figma 等设计工具一样直观地编辑 React 组件。您可以：

- 拖放组件
- 调整元素大小和位置
- 编辑文本和图像
- 使用 Tailwind CSS 应用样式

{/* ![Visual Editor](/images/visual-editor.png) */}

*Onlook 的可视化编辑器提供熟悉的设计工具界面*

## 代码集成

Onlook 与您的代码无缝集成，允许您：

- 实时查看代码变更
- 直接编辑代码
- 导入和导出组件
- 与现有工作流程集成

{/* ![Code Integration](/images/code-integration.png) */}

*即时查看设计变更在代码中的反映*

## AI 辅助

Onlook 的 AI 辅助功能帮助您：

- 根据描述生成组件
- 获取改进建议
- 将设计转换为代码
- 调试问题

{/* ![AI Assistance](/images/ai-assistance.png) */}

*让 AI 帮助您创建和改进设计*

## Figma 到 Onlook

将您的 Figma 设计直接导入 Onlook，以：

- 将设计转换为可工作的 React 组件
- 保持设计保真度
- 使设计具有交互性
- 连接到数据源

{/* ![Figma to Onlook](/images/figma-to-onlook.png) */}

*从 Figma 设计无缝过渡到可工作的代码*
