# UI 概览

Onlook 界面设计直观，便于设计师使用，同时提供强大的 React 组件编辑功能。

## 主要界面区域

Onlook 界面分为以下几个关键区域：

- **画布 (Canvas)**：中央区域，用于查看和交互组件
- **图层面板 (Layers Panel)**：显示组件层级结构，类似于设计工具中的图层
- **属性面板 (Properties Panel)**：编辑所选组件的属性
- **样式编辑器 (Style Editor)**：通过可视化界面修改 Tailwind 样式
- **代码面板 (Code Panel)**：查看和编辑生成的代码
- **AI 聊天 (AI Chat)**：与 AI 交互，帮助生成和修改组件

## 页面导航

- 使用**项目选择器 (Project Selector)** 切换项目
- **文件浏览器 (File Browser)** 允许浏览项目文件
- **面包屑导航 (Breadcrumb Navigation)** 显示当前在组件层级中的位置

## 键盘快捷键

Onlook 提供多种键盘快捷键以加速工作流程：

- **Ctrl/Cmd + S**：保存更改
- **Ctrl/Cmd + Z**：撤销
- **Ctrl/Cmd + Shift + Z**：重做
- **Ctrl/Cmd + C**：复制所选组件
- **Ctrl/Cmd + V**：粘贴组件
- **Delete**：删除所选组件
- **Escape**：取消当前选择
