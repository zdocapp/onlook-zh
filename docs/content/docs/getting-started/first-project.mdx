# 创建你的第一个项目

本教程将指导你在 Onlook 中创建你的第一个项目。

## 步骤 1：创建新项目

1. 打开 Onlook
2. 在欢迎界面点击 "New Project"
3. 选择一个模板或从头开始
4. 输入项目名称
5. 点击 "Create"

## 步骤 2：探索界面

花些时间熟悉 Onlook 界面：

- **Canvas（画布）**：查看和与组件交互的中心区域
- **Layers Panel（图层面板）**：显示组件层级结构
- **Properties Panel（属性面板）**：编辑所选组件的属性
- **Style Editor（样式编辑器）**：修改 Tailwind 样式
- **Code Panel（代码面板）**：查看和编辑生成的代码

## 步骤 3：添加组件

1. 点击工具栏中的 "+" 按钮
2. 从组件库中选择一个组件
3. 将其拖放到画布上
4. 使用属性面板进行自定义

## 步骤 4：样式化组件

1. 在画布上选择一个组件
2. 使用样式编辑器应用 Tailwind 类
3. 调整间距、颜色、排版和其他属性

## 步骤 5：预览和导出

1. 点击 "Preview" 查看项目效果
2. 进行最终调整
3. 点击 "Export" 导出项目
4. 选择导出选项（代码、图像等）
