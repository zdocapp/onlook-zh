# 欢迎使用 Onlook！

Onlook 是一款“面向设计师的 Cursor”，让设计师能够直接在浏览器 DOM 中对 React 和 TailwindCSS 项目进行实时编辑。它在设计与开发之间提供了无缝集成体验。

![Onlook 界面](/images/onlook-interface.png)
*Onlook 直观的设计界面让您能够可视化编辑 React 组件*

## 什么是 Onlook？

Onlook 让您能够：

- 在类浏览器界面中直接编辑设计
- 自动将更改反映到代码中
- 使用 AI 辅助生成和修改代码
- 通过主题系统管理项目样式
- 直接从应用程序部署网站

借助 Onlook，设计师可以掌控实现过程，缩小设计与开发之间的差距。无需深厚的编程知识即可创建生产就绪的代码。
