# 常见问题

### Codesandbox 预览

Codesandbox 可能会要求您确认继续操作。您可以通过切换到旁观模式下的预览模式，点击“是，继续预览”链接，然后切换回来。之后，UI 应该在 iframe 内正确渲染。

![Codesandbox 确认模式](/images/csb-proceed-to-preview.png)

![切换到预览模式](/images/visual-switch-to-preview.png)

### 认证问题

如果您遇到页面重新加载后未认证的问题，请检查您的 Node 版本，安装更新的版本，重新安装依赖项并重启项目。
推荐最低版本为 `v20.16.0` 或最新版本。

避免使用 Node 的 `v20.11.0` 版本，因为该版本过去曾出现过此问题。

```bash
node --version
```

您可能需要删除 Cookie 以清除认证状态。

![删除 Cookie](/images/delete-cookies.png)

### “未找到列”错误

如果您遇到诸如“未找到列”之类的错误，您的数据库可能与模式不同步。可以尝试以下两种方法使其重新同步：

1.  运行 `bun db:push` 将模式推送到数据库。如果存在冲突，可以尝试步骤 2。
2.  运行 `bun db:reset` 以完全重置数据库并重新运行迁移。警告：这将删除数据库中的所有数据。
